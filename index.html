total-map

total-map is a small library for working with types which have finite set of possible values.

Why?

I want to have a Map[K, V] (read from config or created manually) that guarantees totality of apply.

What?

The core of the library is just two things:

AllValuesOf[T]

A typeclass which express all possible values of given type

scala trait AllValuesOf[T] { def toSet: Set[T] }

TotalMap[K, V]

A data structure which guarantees that there exist an entry for every possible key

scala trait TotalMap[K, +V] { def toMap: Map[K, V] def apply(k: K): V }

How?

Install

scala libraryDependencies += "com.github.krever" % "total-map" % "0.0.1+14-c0dad29e"

Import

scala import totalmap._

An then you need to have an instance of AllValuesOf[T]. It can be acquired in one of 2 ways:

Use preexisting instances

scala implicitly[AllValuesOf[Unit]] // res1: AllValuesOf[Unit] = AllValuesOf(()) implicitly[AllValuesOf[Boolean]] // res2: AllValuesOf[Boolean] = AllValuesOf(true,false)

Instances that require materializing big collections are not available, but might come if someone comes with reasonable use case

scala implicitly[AllValuesOf[Int]] // error: could not find implicit value for parameter e: totalmap.AllValuesOf[Int] // implicitly[AllValuesOf[Int]] // ^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Enumeratum

You can get enumeratum support by installing

scala libraryDependencies += "com.github.krever" % "total-map-enumeratum" % "0.0.1+14-c0dad29e"

So when you have an Enum

scala import enumeratum._ sealed trait Animal extends EnumEntry object Animal extends Enum[Animal] { case object Cat extends Animal case object Dog extends Animal override def values: scala.collection.immutable.IndexedSeq[Animal] = findValues }

You can the instances for free

```scala import totalmap.modules.enumeratum._

implicitly[AllValuesOf[Animal]] // res4: AllValuesOf[Animal] = AllValuesOf(Cat,Dog) ```

Create named set

```scala implicit val myStrings = AllValuesOf.set(Set(“a”, “b”, “c”)) // myStrings: AllValuesOf.NamedSet[String] = totalmap.AllValuesOf$$anon$2@30e9877e

implicitly[AllValuesOf[myStrings.Elem]] // res5: AllValuesOf[myStrings.Elem] = AllValuesOf(a,b,c) ```
